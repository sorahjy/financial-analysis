## 0. 游资操盘周期与本文用法

游资=短线主力，完整周期：**建仓(吸筹/埋伏) → 试盘 → 洗盘 → 拉升 → 出货**。
2015 后基本无长庄，多是「热点短庄 + 游资接力」，所以形态以**中短周期**为主。

每个形态给出 4 个字段，便于后续写成匹配函数：
- **信号** 🟢买/吸筹区 · 🟡持有/确认 · 🔴卖/回避
- **量价/K线判据**（可编程，基于日线 OHLCV+换手+成交额）
- **对应雷达特征**（我们已实现的，见 `stock_hot_money_radar.py`）
- **缺口**（需新增的基础特征）

## 1. 可量化基础特征字典（匹配器的积木）

| 代号 | 含义 | 计算 | 现状 |
|---|---|---|---|
| `pos60` | 价格位置 | 收盘在近60日分位 | ✅ 雷达 `_score_position` |
| `vol_ratio` | 量比 | 近5日均量 / 前20日均量 | ✅ `_score_volume_ratio` |
| `turn` | 换手率 | daily_turnover_rate | ✅ |
| `amp` | 振幅 | (high-low)/prev_close | ✅ 调研用过 |
| `cmf` | 收盘买压 | Σ(MFM×量)/Σ量, MFM=((C-L)-(H-C))/(H-L) | ✅ `_score_cmf` |
| `drift20` | 横盘度 | 近20日累计涨跌幅(≈0=横住) | ✅ `_score_absorption` |
| `chip_conc` | 筹码集中度 | 成本分布主峰±7%占比 | ✅ `_chip_metrics` |
| `chip_peak_move` | 筹码峰移动 | 主峰成本价的环比变化方向 | 🔧 需新增(区分洗盘/出货关键) |
| `body/上下影` | K线实体与影线 | 实体=|C-O|/prev_close；下影=(min(O,C)-L)/prev_close；上影=(H-max(O,C))/prev_close | 🔧 需新增 |
| `is_doji` | 十字星 | 实体极小(<0.5%) | 🔧 需新增 |
| `limit_up` | 涨停 | change_pct≥涨停阈值(板块分) | ✅ `_sealed_and_streak` |
| `yizi` | 一字板 | 涨停 且 amp≈0 | ✅ |
| `streak` | 连板数 | 连续涨停天数 | ✅ |
| `breakout` | 平台突破 | close>近N日最高收盘 + 放量 | ✅ `_breakout_trigger` |
| `ma_bull` | 均线多头 | MA5>MA10>MA20 且 close>MA5 | 🔧 需新增(stock_crawl_hot_money 有现成 `compute_tech_signals`) |
| `vol_price_div` | 量价背离 | 价创新高 但 量缩 / 放量但滞涨 | 🔧 需新增 |

> 大部分积木已具备；只缺 **上下影线/十字星、均线关系、量价背离、筹码峰移动方向** 四类，工作量小。

## 2. 吸筹 / 埋伏阶段（🟢 左侧买点预备）

> 共同前提：`pos60` 低（底部/波谷），主力在低成本区收集。

**P1 低位横盘磨人（时间换筹码）**
- 判据：`pos60<0.35` + 连续 N(≥20)日 价格窄幅(`drift20`≈0、振幅持续低) + 换手温和(非<1%的死水) + 小阴小阳。
- 雷达：`_score_position`(低) + `_score_absorption`(横住)；🟢
- 一旦向上突破→量价齐升，是 P11 的前奏。

**P2 低位十字星/长下影/T字反复**
- 判据：底部多日 `is_doji` 或 长下影(`下影 > 2×实体`)反复出现 + 缩量；主力盘中吸货痕迹。
- 缺口：上下影线/十字星。🟢

**P3 隐性收集（打压吸筹·左长阴右长阳"打桩"）**
- 判据：`pos60<0.35`；近8日出现跌幅≥4.5%的大阴，且当日量<此前20日均量的85%；之后最多再破底3%，首次完整收复跌前收盘时确认。
- 确认日量≤此前20日均量的1.30倍，避免把放量追涨混入吸筹；一次下跌只触发一次。🟢

**P4 量增价稳（吸收/努力与结果背离）** ⭐核心
- 判据：`vol_ratio↑`(1.2~2.5) **且** `drift20`≈0(价格横住) **且** `cmf>0`(收盘买压正)。放量却不涨=有人在吸货。
- 雷达：这正是 `_score_divergence` + `_score_cmf`；🟢

**P5 底部形态构筑**
- 判据：圆弧底 / W底 / 头肩底 / 箱体底（局部低点抬高、双底颈线）。
- 缺口：底部形态识别（局部极值序列）。🟢

**P25 低位横盘缩量后启动（全池统一实验买点）**
- 结构：收盘价处于近120日后45%，近60日振幅≤25%、绝对涨跌≤8%，近20日均量不超过更早40日的70%，且60日平滑成交量斜率≤-1%。
- 启动：当日量能≥前20日均量1.5倍，收盘距前20日平台高点不低于-2%。
- 防追高：近5日涨幅≤8%，当日及近3日最大单日涨幅<6%。
- 范围：`leader`、`hotmoney` 与未指定池使用同一判据。样本外显著性仍不足，所以仅作实验标签。🟢

## 3. 试盘阶段（🟡 观察，主力测压力）

**P6 试盘长上影（破平台又缩回）**
- 判据：单日 `high` 突破近 N 日平台高 + **长上影**(`上影>2×实体`) + 次日跌回平台内；测上方抛压/筹码稳定性。
- 缺口：上影线 + 平台。🟡（试盘成功→洗盘→拉升）

**P7 底部异动放量**
- 判据：底部突然单日 大振幅+放量(`amp`高 & `vol_ratio`高)，方向不明；主力试盘。🟡

---

## 4. 洗盘阶段（🟢 上车机会 / 区分出货是关键）

> ⭐ **全清单最重要的判别：洗盘 vs 出货**
> - **洗盘(🟢)**：缩量下跌 + 放量上涨 + **筹码峰不发散**(不破关键均线) → "下跌没人卖、上涨有人买"
> - **出货(🔴)**：放量下跌 + 缩量反弹 + **筹码峰快速下移**

**P8 缩量回踩洗盘（挖坑）**
- 判据：拉升前回调，下跌日 `量缩至近期均量1/2以下` + 上涨日放量 + `chip_conc` 不降 + 不破 MA20/MA60。
- 雷达：`chip_metrics` + 需 `ma_bull`/均线；🟢（最佳跟庄点）

**P9 边拉边洗**
- 判据：`ma_bull`(均线多头发散) + K线阴阳交错 + 成交量规则缩放 + 低点不断抬高。
- 缺口：均线 + 低点抬高序列。🟢

**P10 高换手洗盘（成长赛道）**
- 判据：放量但 `chip_peak_move`≈0(筹码峰不发散) → 区别于出货的筹码峰下移。🟢

---

## 5. 拉升阶段（🟡 超短动量；持有期拉长需防反转）

**P11 放量突破启动（强收集）**
- 判据：`breakout`(放量突破前期平台/箱体, `vol_ratio>1.5`) + 收阳 + 从低/中位起(`pos60`突破前<0.7)。
- 雷达：`_breakout_trigger`（右进左出）。🟡 全历史逐日复测显示双池未来负超额，按追高风险处理。

**P12 连板拉升 / 一字封死**
- 判据：`streak≥2` 连续涨停；`yizi`(一字板)=封死换手骤降(~0.7%)；换手板=盘中有换手。
- 雷达：`_sealed_and_streak`。🟡 复测 2 日动量有效；hotmoney 池 10~40 日转为负超额，不能把超短信号外推成长持有。

**P13 首板卡位（打板·黄金时段）**
- 判据：前期无连板的**首次**涨停 + 早盘封板(10:30前·需分时) + 换手 25-40% + 题材龙头/卡位。
- 缺口：分时封板时间(无分时则用日线近似：首板+换手区间+收盘=最高)。🟡 复测 hotmoney 池 2 日动量有效，优势随持有期衰减。

## 6. 出货阶段（🔴 卖出/回避——数据背书最强的部分）

**P26 获利盘风险**
- 条件一：筹码成本分布估算的获利盘比例≥90%。
- 条件二：5个交易日前获利盘比例<40%，且当前获利盘比例≥60%。
- 任一筹码条件成立后，还须满足当日量能≥此前20日均量1.5倍才命中；量能优先使用换手率、缺失时退回成交量。所有股票池统一计算。P26作为独立强风险提示，可直接触发“出货预警”。🔴

> 我们对太极实业的回测已证：**高位(前涨>20-30%)放量上榜/创高后必回撤**。出货预警是本清单最有实战价值的一段。

**P14 高位放量滞涨** ⭐核心
- 判据：`pos60>0.85` + `vol_ratio`高 + 近5日涨幅低/转负(滞涨) + 长上影。创新高但涨不动。
- 缺口：滞涨(短期涨幅) + 上影。🔴 全历史逐日复测双池 2~40 日负超额，风控有效。

**P15 量价背离**
- 判据：价创新高 但 `量缩`(背离量)；或 高位放天量却滞涨。
- 缺口：`vol_price_div`。🔴

**P16 阴天量 / 假阴量**
- 判据：攻击量后某日 急放近期最大量柱 + 收阴(阴天量)；或随后高开低走但不破前阳(假阴量·二次拉高出货)。🔴 全历史逐日复测双池 2~40 日风控有效。

**P17 倒 V 反转**
- 判据：急涨突破→当日/次日急跌，`放量拉升→放量出货`，盘中倒 V；高位。🔴 全历史逐日复测双池 2~40 日持续负超额。

**P18 顶部 K 线形态**
- 判据：大阴包阳 / 双头 / 头肩顶 / 尾盘跳水次日低开。
- 缺口：顶部形态识别。🔴

**P19 灌压出货（巨量大阴）**
- 判据：高位不计成本砸盘，单日 巨量(`vol_ratio`极高) + 大阴线(实体大、收盘近最低)。🔴 全历史逐日复测双池 2~40 日持续负超额，是幅度最强的风控形态。

**P20 均线放量破位**
- 判据：5/10日均线反复放量跌破且反弹站不回；布林下轨有效击穿。
- 缺口：均线 + 布林。🔴 全历史逐日复测双池 2~40 日负超额；独立形态和复合吸筹模型破位惩罚均保留。

## 7. 落地路线（后续做匹配器）

1. **补 4 类基础特征**：上下影线/十字星、均线(MA5/10/20/60+多头)、量价背离、筹码峰移动方向（`chip_metrics` 已有主峰价，加环比即可）。`stock_crawl_hot_money.py::compute_tech_signals` 已有 MA/RSI/MACD/连板/一字板，可直接复用。
2. **每个 Pxx 写成一个布尔/打分匹配函数**（输入近端 bars，输出命中+强度），复用雷达 `_recent_bars` 与现有子特征。
3. **优先级**（按最新复测的数据背书 + 实战价值）：
   - 先做 🔴出货预警（P14/P16/P17/P19/P20/P22）及负向突破风险（P11）；
   - P6/P7 仅保留为观察形态，不参与有效因子高亮；
   - P12/P13 仅作 2 日超短动量提示，同时明确中长持有的反转风险；
   - P24 作为双池10日吸筹买点高亮；P2/P15/P18只作次级提示，P23及其余无效形态只作状态或研究标签。
4. **验证**：每个形态命中后，用现成 `verify` 框架测「命中→未来 N 日前向收益」的截面 IC / 事件超额，只保留有正向预测力的形态。
