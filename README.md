# financial-analysis
检查：1）基金各个阶段的超额收益。2）基金技术面分析与买卖点推荐。3）基金经理是否变更。

### 1 Requirements

* pip install -r requirements.txt


### 2 基金相关配置与使用
##### 2.1 STEP 1 参数修改

* funds.py 里可以配置具体的基金。

* technical_analysis.py 中可以设置各个指标的参数。

##### 2.2 STEP 2 运行

* bash fund_run.sh

##### 2.3 Outputs
会输出一个html文件 'fund_report.html' 展示：
 * 基金近期超额收益表现。
 * 基金当日的买卖指标和技术面指标。基金的历史买卖点。 
 * 提示经理人事变动情况
